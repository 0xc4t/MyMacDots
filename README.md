### Window Focus (Arrow Keys)

| Key | Action |
|----|-------|
| `Alt + ←` | Focus window west |
| `Alt + ↓` | Focus window south |
| `Alt + ↑` | Focus window north |
| `Alt + →` | Focus window east |

---

### Window Swap

| Key | Action |
|----|-------|
| `Alt + Shift + ←` | Swap window west |
| `Alt + Shift + ↓` | Swap window south |
| `Alt + Shift + ↑` | Swap window north |
| `Alt + Shift + →` | Swap window east |

---

### Window Controls

| Key | Action |
|----|-------|
| `Alt + U` | Toggle floating |
| `Alt + F` | Toggle fullscreen (zoom) |
| `Alt + Q` | Close focused window |

---

### Application Shortcuts

| Key | Action |
|----|-------|
| `Alt + Return` | Open Kitty (single instance) |
| `Alt + P` | Open Flameshot |

---

### Space (Workspace) Navigation

| Key | Action |
|----|-------|
| `Alt + 1..9` | Focus space 1–9 |
| `Alt + 0` | Focus space 10 |

---

### Move Window to Space

| Key | Action |
|----|-------|
| `Alt + Shift + 1..9` | Move window to space 1–9 |
| `Alt + Shift + 0` | Move window to space 10 |

---

### Move Window Between Spaces

| Key | Action |
|----|-------|
| `Alt + Shift + P` | Move window to previous space |
| `Alt + Shift + N` | Move window to next space |
